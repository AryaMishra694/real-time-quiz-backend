## My Contributions

This project was developed as part of a team. The following features reflect my individual contributions to the backend system:

### Authentication
- Implemented user and admin login functionality (`AuthLogin`, `AdminAuthLogin`)

### Quiz Management
- Developed APIs for quiz creation and removal (`QuizCreate`, `QuizRemove`, `AdminQuizCreate`, `AdminQuizRemove`)

### Question Management
- Built functionality to delete and duplicate quiz questions (`QuizQuestionDelete`, `QuizQuestionDuplicate`)

### Player & Session Features
- Implemented player interaction endpoints including:
  - Fetching question details (`GetPlayerQuestionDetail`)
  - Submitting answers (`PlayerSubmitAnswer`)
  - Retrieving session results (`GetQuizSessionResults`)
  - Real-time chat functionality (`SendChat`)

> Note: This repository contains a curated subset of the original project focusing on my backend contributions.
